¿Qué tipo de variables se ingresan a una regresión lineal?

La importancia de conocer la naturaleza de las variables de un Dataset con el que tengamos que trabajar radica en esta sección, dado que no podemos utilizar cualquier tipo de variable en un modelo de regresión lineal.

Para la variable dependiente o a explicar o predicha (y), se es fundamental que sea una variable numérica no categórica ni binaria.

Para las variables independientes (x), se necesita que sean variables numéricas o categóricas convertidas a numéricas.

La explicación rápida de porqué esto debe ser así es simplemente porque el algoritmo de una regresión lineal está construida de esa forma. Si utilizaramos una variable binaria como dependiente, estaríamos hablando de un modelo Logit que es algo totalmente distinto de una regresión lineal. Aunque esto no quiere decir que el modelo de Regresión Lineal no te otorgue salidas si ignoramos lo anterior, sino que va a otorgar salidas pero será un modelo mal planteado.
