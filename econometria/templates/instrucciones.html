Instrucciones para una correcta salida del modelo

  • Las columnas de las bases de datos deben estar correctamente específicados y no deben contar con carácteres como comillas o comillas dobles.
  • La aplicación solo acepta archivos con extensiones .csv, .xls, .xlsx y .txt.
  • Los archivos no deben superar los 3Mb.
  • Las variables deben ser numéricas. (Tanto para las variables dependientes como independientes)
  • La aplicación limpia de manera automática los valores NA.
