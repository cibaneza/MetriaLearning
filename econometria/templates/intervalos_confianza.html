Intervalos de confianza para Beta cero, Beta uno y la varianza

Los intervalos de confianza son una medida de "confiabilidad" ante el error de los estimadores puntuales. Si ya aprendiste calcular intervalos de confianza para la media y la varianza en Estadística Inferencial, en las Regresiones Lineales puedes hacerlos para Beta cero, Beta uno y la Varianza.

Estadístico de prueba

El estadístico t, T-Student, se vuelve el modelo a seguir al momento de calcular intervalos de confianza, dado que a menudo estamos ante muestras y desconocemos el valor poblacional de la varianza además, a largo plazo, este estadístico se vuelve muy similar a la distribución normal, se utilizan principalmente sobre los Betas.

Test de hipótesis

Los test de hipótesis son una gran herramienta de la Estadística Inferencial que nos permite también utilizarlas para las Regresiones Lineales. Generalmente, lo que se busca es que los coeficientes de los Betas sean significativos para el modelo (distintos de cero) a cierto nivel de significancia o error. Para esto, se establece lo siguiente $$\text{Para Beta uno}$$ $$H_0: \beta_1 = 0$$ $$H_1: \beta_1 \neq 0$$ $$\text{Y, para Beta cero}$$ $$H_0: \beta_0 = 0$$ $$H_1: \beta_0 \neq 0$$

Aunque también puedes utilizarlas para buscar un valor concreto, a cierto nivel de significancia. $$\text{Para Beta uno}$$ $$H_0: \beta_1 = 10$$ $$H_1: \beta_1 \neq 10$$ $$\text{Y, para Beta cero}$$ $$H_0: \beta_0 = 5$$ $$H_1: \beta_0 \neq 5$$

Error Estándar Beta uno

Entonces, la definición del estadístico t se presenta como $$t_c = \frac{\widehat{\beta}_1-\beta_1}{\sqrt {MSRes/(\sum_{i=1}^n x^2_i-n\bar{x}^2)}}$$ $$\text{t calculado:} \quad t_c$$ Para Beta uno. Entonces, el error estándar para Beta uno es $$se(\widehat{\beta}_1) = \sqrt {\frac{MSRes}{\sum_{i=1}^n x^2_i-n\bar{x}^2}} $$ Recordando que MSRes corresponde a $$MSRes = \frac{SSRes}{n-2} = \sigma^2$$ Por lo que, otra forma de escribir el error estándar para Beta uno sería $$se(\widehat{\beta}_1) = \sqrt {\sigma^2 (\frac{1}{\sum_{i=1}^n x^2_i-n\bar{x}^2})}$$

Error Estándar Beta cero

Entonces, la definición del estadístico t se presenta como $$t_c = \frac{\widehat{\beta}_0-\beta_0}{\sqrt {MSRes(\frac{1}{n}+\frac{\bar{x}^2}{\sum_{i=1}^n x^2_i-n\bar{x}^2})}}$$ Otra forma de escribir el error estándar para Beta cero sería $$se(\widehat{\beta}_0) = \sqrt {\sigma^2 (\frac{1}{n}+\frac{\bar{x}^2}{\sum_{i=1}^n x^2_i-n\bar{x}^2})}$$

Región de Rechazo

En ambos casos, la hipótesis nula (H_0) se rechaza si $$|{t_c}|>t_{1-\frac{\alpha}{2}, n-2}$$ En caso de contar con el p-value (valor p), se rechaza la hipótesis nula si $$p_{value} < \alpha$$

Intervalos de confianza

Si los errores distribuyen normal e independientes, entonces: $$\frac{\widehat{\beta}_1-\beta_1}{se(\widehat{\beta}_1)} \quad \text{y} \quad \frac{\widehat{\beta}_0-\beta_0}{se(\widehat{\beta}_0)} \sim t_{(1-\frac{\alpha}{2}, n-2)}$$

Entonces, un intervalo de confianza para la pendiente Beta uno se determina con $$\widehat{\beta}_1-t_{(1-\frac{\alpha}{2}, n-2)}se(\widehat{\beta}_1) \le \beta_1 \le \widehat{\beta}_1+t_{(1-\frac{\alpha}{2}, n-2)}se(\widehat{\beta}_1)$$

Entonces, un intervalo de confianza para la ordenada al origen o coeficiente de posición Beta cero se determina con $$\widehat{\beta}_0-t_{(1-\frac{\alpha}{2}, n-2)}se(\widehat{\beta}_0) \le \beta_0 \le \widehat{\beta}_0+t_{(1-\frac{\alpha}{2}, n-2)}se(\widehat{\beta}_0)$$

Un intervalo de confianza para la varianza distribuye chi cuadrada, con n-p (n-2, para una RLS) grados de libertad. Entonces $$\frac{(n-2)MSRes}{\chi^2_{(\frac{\alpha}{2}, n-2)}} \le \sigma^2 \le \frac{(n-2)MSRes}{\chi^2_{(1-\frac{\alpha}{2}, n-2)}}$$
